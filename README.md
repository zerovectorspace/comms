# comms

An client to client end-to-end encrypted messaging application with text rendered in OpenGL. Currently in development.

## Prerequisites

- C++17 compliant compiler
- OpenGL libraries
- GLEW
- SDL2
- CMake

## Building


``` bash
$ cd ~
$ git clone <repo>
$ cd <repo>
$ cmake .
$ make main
```

## Features

[x] OpenGL text rendering
[] Multi-window positioning
[] Settings
[] Sockets
[] Protocol
[] Encryption layer
[] STUN server
[] UDP hole punching
